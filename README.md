Mercearia Gertrudes
======================

Exemplo de uso de ADO.Net com ASP.Net MVC 4, criando projetos que demonstra o cadastro e proteção das paginas com sistema de login simples.

## Conteúdo ##
1. Classe de Contexto, criado os comandos para consumir um banco de dados usando ADO.Net.
2. Classe de Aplicação, criado para gerar as Query SQL para a classe de Contexto.
3. Controller consome as classes de Aplicação.
